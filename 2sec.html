Typographie

Tous les titres et sous-titres sont accessibles avec les balise de <h1> à <h6>
Exemple :

Titre h1, Sous-titre

Titre h2, Sous-titre

Titre h3, Sous-titre

Titre h4, Sous-titre

Titre h5, Sous-titre
Titre h6, Sous-titre

Code : 

        <h1>Titre h1, <small>Sous-titre</small></h1>

        <h2>Titre h2, <small>Sous-titre</small></h2>

        <h3>Titre h3, <small>Sous-titre</small></h3>

        <h4>Titre h4, <small>Sous-titre</small></h4>

        <h5>Titre h5, <small>Sous-titre</small></h5>

        <h6>Titre h6, <small>Sous-titre</small></h6>

Puce

Il existe différent type de puce lors de l'affichage de liste. Elles sont utilisable à l'aide de la combinaison de balise <ul><li></li></ul>

Exemple 1:
    Utilisation de puces standard
  • Exemple 1
  • Exemple 2
Code:
        <ul>

          Utilisation de puces standard

          <li>Exemple 1</li>

          <li>Exemple 2</li>

        </ul>
Exemple 2:
    Utilisation des puces "square"
  • ça marche
  • ça marche
Code:
        <ul class="square">

          Utilisation des puces "square"

          <li>ça marche</li>

          <li>ça marche</li>

        </ul>
Exemple 3:
    Utilisation de "unstyled"
  • Does it works ?
  • Est-ce que ça marche ?
Code:
        <ul class="unstyled">

          Utilisation de "unstyled"

          <li>Does it works ?</li>

          <li>Est-ce que ça marche ?</li>

        </ul>

Les balises textes

Il est possible de mettre en forme le texte de diverses manières à l'aide de plusieurs balises

Exemple:
  • Pour barrer
  • Pour attirer l'attention
  • Pour marquer les sections insérées
  • Survolez le titre
Code:
        <del>Pour barrer</del>

        <mark>Pour attirer l'attention</mark>

        <ins>Pour marquer les sections insérées</ins>

        <abbr title="Titre survolé">Survolez le titre</abbr>

Table

    Les tableaux sont utilisable à l'aide d'une succession de balise <table><tr><th></th></tr><tr><td></td></tr></table>.
  • <table> est la balise servant à créer un tableau.
  • <tr> est la balise servant à rajouter une ligne à ce tableau.
  • <td> est la balise servant à ajouter des cellules dans le tableau.
  • <th> a la même fonction que la balise <td> mais sert uniquement pour les titres.
Exemple 1:
Tableau par défaut
Nombre d'utilisateur par système d'exploitation
Ubuntu OSX Windows 10
10 1876 1886
7 592 599
17 2468 2485
Code:
        <table>

          <caption>Nombre d'utilisateur par système d'exploitation</caption>

          <tr>

            <th>Ubuntu</th>

            <th>OSX</th>

            <th>Windows 10</th>

          </tr>

          <tr>

            <td>10</td>

            <td>1876</td>

            <td>1886</td>

          </tr>

          <tr>

            <td>7</td>

            <td>592</td>

            <td>599</td>

          </tr>

          <tr>

            <td>17</td>

            <td>2468</td>

            <td>2485</td>

          </tr>

        </table>

Tableau "alternate"

Bouton

Il est possible de créer un bouton à l'aide des balise <button> ou <a> et des classes de bouton.

Bouton standard
Bouton Bouton bleu Petit bouton rouge Gros bouton jaune
Code:
        <a class="btn" href="#">"Bouton"</a>

        <a class="btn btn-blue" href="#">"Bouton bleu"</a>

        <a class="btn btn-red btn-small" href="#">"Petit bouton rouge"</a>

        <a class="btn btn-yellow btn-large" href="#">"Gros bouton jaune"</a>
Bouton plat
Pour obtenir des boutons plat, il suffit d'ajouter "btn-flat" dans la classe de la balise
Bouton Bouton bleu Petit bouton rouge Gros bouton jaune
Code:
        <a class="btn btn-flat" href="#">Bouton</a>

        <a class="btn btn-flat btn-blue" href="#">Bouton bleu</a>

        <a class="btn btn-flat btn-red btn-small" href="#">Petit bouton rouge</a>

        <a class="btn btn-flat btn-yellow btn-large" href="#">Gros bouton jaune</a>
Bouton extra-large et bouton "fancy"
Pour obtenir un bouton extra-large, il suffit d'ajouter "btn-xtra-large" dans la classe de la balise
Bouton bleu
Pour obtenir un bouton "fancy", il suffit d'ajouter "btn-fancy" dans la classe de la balise
Code:
        <a class="btn btn-xtra-large btn-blue" href="#">Bouton bleu</a>

        <button class="btn btn-xtra-large btn-blue" href="#">Bouton bleu</button>

        <button class="btn btn-fancy btn-yellow">Bouton fancy jaune</button>

        <button class="btn btn-fancy btn-red">Bouton fancy rouge</button>

        <button class="btn btn-fancy btn-blue">Bouton fancy bleu</button>

        <button class="btn btn-fancy">Bouton fancy</button>

Alertes

Les alertes permettent de signaler des danger, un succès, des informations ou un avertissement à l'aide des classes "alert".

Exemple:
Attention adresse invalide.
Félicitation vous avez gagné un téléviseur.
Attention au changement d'heure le week-end prochain !
Veuillez vérifier votre adresse mail !
Code:
        <div class="alert alert-danger">Attention adresse invalide</div>

        <div class="alert alert-success">Félicitation vous avez gagné un téléviseur</div>

        <div class="alert alert-information">Attention au changement d'heure le week-end prochain</div>

        <div class="alert alert-avertissement">Veuillez vérifier votre adresse mail</div>

Jumbotron

Le jumbotron est réalisable à l'aide de la classe "jumbotron".

Exemple:

Titre Jumbotron

Ceci est un paragraphe de Jumbotron, avec, plusieurs lignes parce qu'il faut voir si le Jumbotron gère les lignes correctement.

Code:
        <div class="jumbotron span-4 offset-2">

          <h1>Titre Jumbotron</h1>

            <p>

            Ceci est un paragraphe de Jumbotron, avec,

            plusieurs lignes parce qu'il faut voir si le Jumbotron gère les lignes correctement.

            </p>

          <button class="btn">Test</button>

        </div>

Barre de navigation

Pour obtenir une barre de navigation il suffit d'appliquer à la balise <ul> la classe "navbar".

La gestion des onglets actifs ou inactifs se fait à l'aide des classes "active" et "disabled" sur les balises <li>.

Exemple:
  • Home
  • Liens
  • Contact
  • Produit
  • Test
  • Test2
Code:
        <ul class="navbar">

          <li class="active"><a>Home</a></li>

          <li><a href="#">Liens</a></li>

          <li><a href="#">Contact</a></li>

          <li><a href="#">Produit</a></li>

          <li><a href="#">Test</a></li>

          <li class="disabled"><a>Test2</a></li>

        </ul>
